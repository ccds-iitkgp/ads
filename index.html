Workshop on Accelerated Data Science 2021

The workshop covers state-of-art theory and technologies for accelerated machine learning as well as deep learning algorithms. The fundamentals of GPU computing with introduction to machine learning and deep learning algorithms are presented. Several technologies and platforms for GPU acceleration of these algorithms are covered.
